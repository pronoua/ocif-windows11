# OCIF Paint.NET Plugin

Плагін для [Paint.NET](https://www.getpaint.net/) що дозволяє відкривати, редагувати та зберігати файли у форматі [OCIF](https://github.com/IgorTimofeev/OCIF) (`.pic`) — формат зображень для [OpenComputers](https://github.com/MightyPirates/OpenComputers) мода для Minecraft.

Підтримувані версії OCIF: **v6, v7, v8**

---

## Вимоги

- Paint.NET **5.1+**
- .NET SDK 9.0+ (тільки для збірки)

---

## Встановлення

1. Зберіть DLL:
```cmd
cd ocif-paintnet
dotnet build OcifPlugin.csproj -c Release
```

2. Скопіюйте в папку плагінів Paint.NET:
```cmd
copy "bin\Release\net9.0-windows\OcifFileType.dll" "C:\Program Files\paint.net\FileTypes\"
```

3. Перезапустіть Paint.NET.

Після встановлення `.pic` файли з'являться в списку форматів у діалогах відкриття та збереження.

---

## Видалення

Видаліть `OcifFileType.dll` з папки `C:\Program Files\paint.net\FileTypes\` і перезапустіть Paint.NET.

---

## Відкриття файлів

При відкритті `.pic` кожен символ OCIF рендериться у комірку **8×16 пікселів**:

| Тип символу | Рендер |
|-------------|--------|
| Брайль (U+2801–U+28FF) | Кожна точка = блок 4×4 пікселів |
| Текстові символи | Глиф з вбудованого шрифту OC Unifont |
| Пробіл (U+0020) | Заливка фоновим кольором |
| Прозорі комірки | Прозорі пікселі |

---

## Збереження файлів

При збереженні через **Файл → Зберегти як** з'являється діалог з налаштуваннями конвертації:

### Дизеринг
| Параметр | Діапазон | Опис |
|----------|----------|------|
| Використовувати дизеринг | вкл/викл | Floyd-Steinberg дизеринг для кращої якості |
| Інтенсивність дизеринга | 0.0 – 1.0 | Сила дизеринга (активно тільки з галочкою) |

### Корекція кольору
| Параметр | Діапазон | Опис |
|----------|----------|------|
| Яскравість | -1.0 – 1.0 | Зміщення яскравості |
| Контраст | -1.0 – 1.0 | Підсилення/ослаблення контрасту |
| Насиченість | -1.0 – 1.0 | Насиченість кольорів |
| Гамма-корекція | 0.2 – 3.0 | Нелінійна корекція яскравості |

### OCIF-специфічні параметри
| Параметр | Діапазон | Опис |
|----------|----------|------|
| Режим символів | Braille / SemiPixel | Braille = брайль (2×4), SemiPixel = напівблок ▄ (1×2) |
| Поріг брайля | 0.0 – 1.0 | 0.5 = стандарт, менше = більше fg точок |
| Медіанне усереднення | вкл/викл | Альтернативний вибір bg/fg по медіані яскравості |
| Мін. різниця кольорів | 0.0 – 1.0 | Схожі кольори замінюються пробілом (0 = викл) |

### Режими символів

**Braille** — кожна комірка кодується брайлем (U+2800–U+28FF). Роздільна здатність 2×4 OCIF-пікселя на комірку. Краще для деталізованих зображень.

**SemiPixel** — кожна комірка кодується символом `▄`. Верхня половина = bg, нижня = fg. Роздільна здатність 1×2 OCIF-пікселя на комірку. Краще передає кольори.

## Прозорість

Прозорість повністю підтримується:
- Прозорі пікселі в Paint.NET → прозорі комірки в OCIF
- Часткова прозорість в комірці → брайль символ де прозорі точки = bg, непрозорі = fg, bg прозорий (`alpha=255`)
- При відкритті прозорі комірки відображаються як прозорі пікселі

---

## Вбудований шрифт

Плагін містить вбудований шрифт **OC Unifont** (font.hex) стиснутий zlib+base64 прямо в DLL. Зовнішніх залежностей немає.

---

## Технічні деталі

- Збережені файли завжди у форматі **OCIF v8**
- Відкриваються файли форматів v6, v7, v8
- Кожна комірка 8×16 пікселів = один символ OCIF (2×4 OCIF-пікселя)
- Палітра: 240 кольорів OC від `0x000000` до `0xFFFFFF`
